# MeetStream Labs

A collection of open source use cases and integrations built on the [MeetStream API](https://meetstream.ai).

Browse, clone, and run any example directly — no setup beyond your MeetStream API key.

## What's inside

Each folder is a standalone use case. Navigate into any one and follow its README to get started.

## Prerequisites

- A MeetStream account and API key → [Get one here](https://app.meetstream.ai)

Have a use case to share? Open a pull request.
